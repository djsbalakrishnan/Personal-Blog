### [Codeshare](https://code.dhirajbalakrishnan.dev)

**A simple way to share code during lectures.**

I built Codeshare for my lectures. I usually ask students to share code in the chat window, and it quickly becomes a hassle to copy, paste, and debug when formatting breaks.

With this, I can share code with students and they can share code back with me during lecture time, without the formatting mess.

**Best for:** instructor-student code sharing, quick debugging sessions, and clean code handoffs over chat.

### [Pomodoro](https://pomodoro.dhirajbalakrishnan.dev)

**A no-distraction timer for focused work.**

The Pomodoro Technique is simple: work for 25 minutes, break for 5, and repeat. Most Pomodoro apps add too many features and become the distraction themselves. This one does not. It is a single-purpose timer: start it, focus, and stop when it rings.

I built this because I kept losing track of time during deep work sessions. A visible countdown and a hard stop improve my time awareness and prevent those 3-hour blur sessions where I forget to eat.

**Best for:** writing, coding, studying, or any task that benefits from focused sprints and enforced breaks.

&rarr; [pomodoro.dhirajbalakrishnan.dev](https://pomodoro.dhirajbalakrishnan.dev)

GitHub Link: [djsbalakrishnan/pomodoro](https://github.com/djsbalakrishnan/pomodoro)

### [Memento Mori](https://www.dhirajbalakrishnan.dev/memento-mori/)

**A dashboard reminding you that time is finite.**

I got introduced to *Memento Mori* through Stoic philosophy and *Meditations* by Marcus Aurelius.

This tool is built for me, not for anyone else. I assume I will live until 80, and this dashboard maps life in weeks so I can see the bigger picture clearly.

For me, this is not about fear. It helps me avoid overthinking and stay grounded in what matters today.


**Best for:** a periodic reality check. Open it when I am deciding how to spend my time.
